Question-1: What are the different ways to select an element in the DOM?

Answer: We can use getElementById(), getElementsByClassName(), querySelector() etc to select an element in the DOM

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML means the HTML tags that are present in the selected element, innerText are the texts which are visible in the website, and textContent gets all text content without any style.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique where we use a single event listener to a parent element to handle events for child elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling means that an event starts at the target element and bubbles up to its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: We use document.createElement() to create elements, appendChild() to add the created elements with a parent div, and remove them with removeChild()
